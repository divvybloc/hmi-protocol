## Architecture

HMI is structured across three primary layers:

### ❤️ Energy Layer (Physical)
- Healthcare systems
- Devices, sensors, real-world inputs

### 💚 Capital Layer (Economic)
- Payments, DeFi, insurance
- Token exchange (QCX™)

### 💙 Intelligence Layer (Cognitive)
- AI agents (23see™)
- Data models, decision systems
